Portfolio projects

NeuralNet for manufacturing control data

Project part of a Kaggle Machine Learning competition of my Master's degree. I trained a neural network with 900.000 instances of 33 features each using Tensorflow to predict a binary output. I applied feature engineering and feature interaction to spot patterns and transform the data to achieve 97.4% accuracy.

Labyrinth Challenge

The goal of this challenge is to design an efficient algorithm that returns the minimum number of moves required by an object to traverse the input grid. The grid is a matrix with some blocked cells and the object is a rod composed of 3 adjacent cells...

Movie recommender engine

Recommender engine based on the movielens dataset. I implemented 3 approaches to recommend movies: Non personalized, content based with an implementation of a Naive Bayes spam filter and finally collaborative filtering (item-item and user-user).

Clustering election data

I performed a clustering of election results per region. I used DS techniques, including high dimensional outlier detection with Mahalanobis distance, standardization, K-Means clustering and validation with prediction-strength method.

Image classification on the CIFAR-10 dataset

Image classificacion on the CIFAR-10 dataset with the fastai library using resnet34 and resnet50 pretrained CNN. Applied Data augmentation techniques such as mixup.

Flight deals search engine

Working with several APIs. Read flight prices to several destinations from a Google sheet file using the Sheety API. Use the Tequila API to fetch flight data, searching for flights that meet criteria. Send an email if the fetched price is lower than the one in the sheet and update it.

Full-Featured Blog Web APP

Personal Blog web app built with Flask, HTML, JS, CSS and bootstrap. Users, posts and post messages are stored in a SQLite db. Allowing user authentication to login and post messages, password encryption and admin privileges to publish/delete posts. Data is dynamically rendered from the database.

Top films Web APP

Built with Flask. Backend functionality allows to add/delete films, edit rating and review of a particular film. User types the name of the film and the backend calls the moviedb API to fetch the data and store it in a SQLite db. Films are rendered dynamically from the db in order according to rating.

Cafe & Wifi API REST

Built a REST API Service to request data stored in a Coffee Shops SQLite db. GET a random coffee shop, GET all coffee shops, search all shops at a particular location, POST a new shop (insert to the database), PATCH coffee price at a particular shop given its database id and DELETE a coffee shop from the db.
